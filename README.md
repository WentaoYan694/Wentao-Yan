# Connect 4 Game (Python)

## Description

This project is a command-line implementation of the classic Connect 4 game, developed in Python using object-oriented programming (OOP).

It supports two-player gameplay on a customizable board. The program includes core functionalities such as piece placement, turn switching, input validation, and automatic winner detection across horizontal, vertical, and diagonal directions.

The system is structured using modular classes:
- Column: manages individual columns and piece placement  
- GameBoard: represents the board and manages game state  
- Connect4Game: controls gameplay flow and win conditions  

This project demonstrates object-oriented design, data structure organization, and algorithmic implementation of game logic.
